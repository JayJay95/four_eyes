# FourEyes

A gem to implement the maker-checker principle. The 4-eyes principle.



## Installation

Add this line to your application's Gemfile:

    gem 'four_eyes'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install four_eyes
